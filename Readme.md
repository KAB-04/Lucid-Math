# AI Math Tutor

An AI-powered adaptive mathematics tutoring system designed to provide personalized mathematics learning support for JHS, SHS, and university-entry learners.

## Objective

The system aims to identify learner weaknesses, create personalized learning pathways, and provide adaptive mathematics instruction using Artificial Intelligence.

## Main Features

- Diagnostic mathematics assessment
- Learner modeling
- Adaptive learning recommendations
- AI mathematics tutor
- Progress tracking

## Technology Stack

Backend:
- C#
- ASP.NET Core

Database:
- PostgreSQL

Frontend:
- React

AI:
- Gemini API
